## 14. 자주 묻는 질문 (FAQ)

**Q: 정비 중인 설비 상태가 자동으로 바뀌나요?**
A: 네, 정비를 시작하면 설비 상태가 자동으로 '수리중'으로 변경되고, 완료하면 '정상'으로 복구됩니다.

**Q: PM 일정을 놓치면 어떻게 되나요?**
A: 예정일이 지나면 자동으로 '지연' 상태가 되며, 대시보드와 알림에서 확인할 수 있습니다.

**Q: 부품 재고가 부족하면 정비를 시작할 수 없나요?**
A: 부품 등록 시 재고 경고가 표시되지만, 정비 시작은 가능합니다. 단, 재고 관리를 위해 주의가 필요합니다.

**Q: 언어를 변경하려면 어떻게 하나요?**
A: 로그인 화면 또는 상단 프로필 메뉴에서 언어를 변경할 수 있습니다.
